# Source Code

This folder contains the core implementation of the project.

## Structure (planned)

- data_loader.py → dataset loading and preprocessing
- bias_injection.py → functions for introducing spurious correlations
- models/ → implementations of different models (Logistic Regression, LSTM, DistilBERT)
- train.py → training pipeline
- evaluate.py → evaluation and metrics

## Notes
- The main entry point for the project is `main.py` in the root directory
- This folder will be expanded as the project develops
